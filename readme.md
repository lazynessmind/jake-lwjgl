# Hello Triangle

Hello Triangle using LWJGL and Jake.

# Build:

First you need to build jake and move the file to this folder. Check [Jake](https://github.com/lazynessmind/jake) to learn how to.

```bash
./jake -b .
java -jar lwjgl-1.0.0.jar
```
